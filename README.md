# CardioLab AI Agent

## Features
- Chat with memory like ChatGPT
- PIV Analysis tool
- TGT Results tool
- uPAD CKD tool
- PubMed live search

- Brain: Llama 3.3 70B via Groq free
- Base: Biomni Stanford Apache 2.0
- UI: Gradio
- Host: Hugging Face Spaces free
