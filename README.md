# CardioLab AI Agent
## SJSU Biomedical Engineering

Live: https://huggingface.co/spaces/Saicharan21/CardioLab-AI
Code: https://github.com/pranatechsol/Cardio-Lab-Ai

## Features
- Chat memory like ChatGPT plus PubMed search
- Voice assistant Groq Whisper
- Paper search PubMed plus Semantic Scholar
- PIV CSV upload 4 separate charts plus AI analysis
- TGT CSV upload blood charts plus thrombogenicity assessment
- uPAD photo upload instant CKD diagnosis
- AI image generation FLUX.1

## Structure
- app.py — Main app v27
- cardiolab_tools/ — 4 custom tools
- cardiolab_knowhow/ — Lab protocols
- test_data/ — Sample CSV files
- versions/ — Version history v23 to v27

## Tech Stack
- Llama 3.3 70B via Groq — free
